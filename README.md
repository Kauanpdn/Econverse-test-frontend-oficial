# 🚀 Econverse - Teste Frontend Oficial

Projeto desenvolvido como teste técnico para vaga de Desenvolvedor Frontend, utilizando **React + TypeScript + Vite**.

O objetivo foi construir uma aplicação moderna, organizada e baseada em boas práticas de componentização, consumo de API e estruturação de layout.

---

## 🛠️ Tecnologias Utilizadas

* React
* TypeScript
* Vite
* CSS / SCSS
* API REST (consumo de dados)
* Componentização reutilizável

---

## 📂 Estrutura do Projeto

```
src/
 ├── components/
 ├── pages/
 ├── services/
 ├── styles/
 ├── App.tsx
 └── main.tsx
```

A estrutura foi organizada para manter separação de responsabilidades, facilitando manutenção e escalabilidade.

---

## ⚙️ Funcionalidades Implementadas

* ✅ Banner principal
* ✅ Seção de categorias
* ✅ Vitrine de produtos
* ✅ Consumo de API
* ✅ Renderização dinâmica de produtos
* ✅ Modal de visualização de produto
* ✅ Componentes reutilizáveis
* ✅ Layout alinhado ao Figma proposto

---

## ▶️ Como rodar o projeto

### 1️⃣ Clonar o repositório

```bash
git clone https://github.com/Kauanpdn/Econverse-test-frontend-oficial.git
```

### 2️⃣ Acessar a pasta

```bash
cd Econverse-test-frontend-oficial
```

### 3️⃣ Instalar dependências

```bash
npm install
```

### 4️⃣ Rodar o projeto

```bash
npm run dev
```

O projeto estará disponível em:

```
http://localhost:5173
```

## 🧠 Decisões Técnicas

* Utilização do **Vite** para melhor performance no desenvolvimento.
* Uso de **TypeScript** para tipagem estática e maior segurança no código.
* Separação por componentes para facilitar reutilização.
* Estrutura pensada para escalar para um e-commerce real.
